# EVINFLOW

EVINFLOW es una aplicación web desarrollada como proyecto académico para la gestión de clientes, proyectos, presupuestos y tareas de freelancers y pequeños negocios.
El objetivo del sistema es centralizar la información de un negocio en un solo lugar, permitiendo administrar clientes, organizar proyectos, controlar presupuestos y hacer seguimiento a las tareas pendientes de forma sencilla.

## Funcionalidades

- Landing Page
- Registro e inicio de sesión
- Dashboard
- Gestión de clientes
- Gestión de proyectos
- Gestión de presupuestos
- Gestión de tareas
- Configuración de perfil
- Panel de administración (en desarrollo)
